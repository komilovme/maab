Puzzle 1:
Max:10 ball
Mode:Easy
Berilgan s nomli satr mavjud. Undagi birinchi takrorlanmaydigan belgini toping va uning indeksini qaytaring.

Agar bunday belgi mavjud bo‘lmasa, -1 ni qaytaring.
Example 1:
Input: s = "appleap"
Output: 5

Example 2:
Input: s = "abcabc"
Output: -1


Puzzle 2:
Max:15 ball
Mode:Medium
Berilgan nums nomli, uzunligi n bo‘lgan massiv mavjud. Sizdan massiv ichidagi asosiy (majority) elementni qaytarish talab qilinadi.
Asosiy element — bu massivda (n / 2) martadan ko‘p uchraydigan elementdir.

Siz shuni faraz qilishingiz mumkin-ki, massivda har doim bitta asosiy element mavjud
Example 1:
Input: nums = [3,2,3]
Output: 3

Example 2:
Input: nums = [2,2,1,1,1,2,2]
Output: 2


Puzzle 3:
Max:15 ball
Mode: Medium
nums nomli massiv berilgan bo‘lib, unda [0, n] oraliqdagi n ta turli (takrorlanmaydigan) sonlar mavjud.

Sizdan shu oraliqda bo‘lishi kerak bo‘lgan, lekin massivda yo‘q bo‘lgan yagona sonni topib, qaytarish talab qilinadi.
Example 1:
Input: nums = [9,6,4,2,3,5,7,0,1]
Output: 8

Example 2:
Input: nums = [0,1,4,2,5,3]
Output: 6



Puzzle 4:
Max:20 ball
Mode: Hard
Fibonacci sonlari, odatda F(n) bilan belgilanadi, Fibonacci ketma-ketligi deb ataladigan ketma-ketlikni tashkil qiladi. Bu ketma-ketlikda har bir son o‘zidan oldingi ikkita sonning yig‘indisiga teng bo‘ladi va u 0 va 1 dan boshlanadi. Shu bilan birga, sizga n soni beriladi siz berilgan n sonidan kichik barcha Fibonacci sonlarini chiqarishingiz kerak

Example:
Input: n=10
Output: [0, 1, 1, 2, 3, 5, 8]


Puzzle 5:
Max:25 ball
Mode:Hard
Sizga butun sonlardan iborat nums massivi berilgan. Shunday answer massivini qaytaringki, uning har bir answer[i] elementi nums massivining nums[i] dan tashqari barcha elementlarining ko'paytmasiga teng bo'lsin.

Example 1:
Input: nums = [1,2,3,4]
Output: [24,12,8,6]

Example 2:
Input: nums = [-1, 1, 0, -3, 3]
Output: [0, 0, 9, 0, 0]
